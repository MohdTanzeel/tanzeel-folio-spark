# Vanilla HTML/CSS/JS Portfolio Website

This is a pure HTML/CSS/JavaScript version of your portfolio website.

## Files Included
- `index.html` - Main HTML structure
- `styles.css` - All styling and design system
- `script.js` - Interactive functionality
- `README.md` - This file

## How to Use

### Option 1: Open Locally
1. Download all files to a folder on your computer
2. Open `index.html` in any modern web browser
3. That's it! The website will work offline

### Option 2: Deploy Online
You can host this website on any of these free platforms:

**GitHub Pages:**
1. Create a GitHub repository
2. Upload all files
3. Go to Settings > Pages
4. Select main branch as source
5. Your site will be live at `username.github.io/repository-name`

**Netlify:**
1. Create account at netlify.com
2. Drag and drop the folder
3. Your site will be live instantly

**Vercel:**
1. Create account at vercel.com
2. Import the folder
3. Deploy with one click

## Customization

### Update Your Information
Open `index.html` and search for:
- `your.email@example.com` - Replace with your email
- `+1 (555) 123-4567` - Replace with your phone
- `Your City, Country` - Replace with your location
- `yourusername` - Replace with your GitHub username
- Update project details, experience, and skills

### Change Colors
Open `styles.css` and modify the CSS variables at the top:
```css
:root {
    --primary: hsl(189, 94%, 43%);  /* Main color */
    --accent: hsl(43, 96%, 56%);    /* Accent color */
    /* etc... */
}
```

### Change Hero Background
In `styles.css`, find `.hero` and update the background image URL:
```css
.hero {
    background: linear-gradient(...), url('YOUR_IMAGE_URL');
}
```
